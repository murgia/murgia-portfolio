Projects

Kilowatt Calculator

Kilowatt calculator is an app with the back-end built in Rails and the front-end built using Angular. Users have the ability to calculate their yearly electricity use and have access to the Energy Information Agency API to calculate projected cost. The app was deployed on Heroku.

SwoleMate

Swolemate is a one-page app built using Express and Node.js. The API was built with MongoDB serving JSON. Users have the ability to create their own workouts as well as access a third-party API. User authentication was done using Passport and the application was deployed on Heroku.

Taskager

Built in Ruby on Rails in one week, Taskager is a task management web application that demonstrates full CRUD functionality with complex many to many relationships. User authentication was done using the Devise gem and the app was deployed on Heroku.
